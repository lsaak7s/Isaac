## 📌 Sobre o projeto

O **Gerador de CSS com IA** é uma ferramenta web que transforma descrições em linguagem natural em código CSS funcional. Basta digitar o que você quer — como *"botão azul arredondado com sombra suave"* — e a IA gera o CSS correspondente automaticamente.

Sem frameworks, sem dependências pesadas. Apenas HTML, CSS e JavaScript puro.

---

## ✨ Funcionalidades

- 💬 **Geração por prompt** — descreva o estilo em texto livre e receba o CSS
- ⚡ **Resultado instantâneo** — código gerado em segundos
- 📋 **Copiar com um clique** — botão de cópia integrado ao resultado
- 🌐 **100% no navegador** — sem necessidade de instalação ou backend
- 🎯 **Simples e direto** — interface minimalista focada na produtividade

---

## 🚀 Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Isaak7s/Gerador-CSS.git
   ```

2. Abra o arquivo `index.html` no navegador — ou acesse pela GitHub Pages.

3. Digite uma descrição no campo de texto, por exemplo:
   ```
   card com fundo escuro, bordas arredondadas e sombra roxa
   ```

4. Clique em **Gerar CSS** e copie o resultado!
